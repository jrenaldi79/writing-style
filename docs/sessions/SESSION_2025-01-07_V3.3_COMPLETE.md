### 2. Authority Signals from Comments

**Pattern recognition:**
```python
authority_words = ['best', 'leader', 'expert', 'mentor', 'one of a kind']

for post in posts:
    for comment in post['top_comments']:
        if any(word in comment['comment'].lower() for word in authority_words):
            # This post demonstrates authoritative voice
            # Use for persona examples
```

**Result:** Know which posts establish credibility

### 3. Content Balance Analysis

**Original vs Curator:**
```python
original = [p for p in posts if p['post_type'] == 'original']
reposts = [p for p in posts if p['is_repost']]

if len(original) > len(reposts):
    persona = "Creator Voice (primarily original content)"
else:
    persona = "Curator Voice (thoughtful commentary on others' work)"
```

### 4. Network Collaboration Patterns

**Who does he amplify?**
```python
from collections import Counter

tagged = Counter(p['name'] for post in posts 
                 for p in post['tagged_people'])

top_collaborators = tagged.most_common(5)
# Example: [('Logan LaHive', 8), ('Startup Founder X', 5)]

# Add to persona: "Frequently collaborates with Logan LaHive"
```

### 5. Topic Resonance Discovery

**What gets engagement?**
```python
# Group by topic (from embedded links/tags)
startup_posts = [p for p in posts if 'startup' in p['text'].lower()]
tech_posts = [p for p in posts if 'AI' in p['text'] or 'product' in p['text']]

avg_likes_startup = sum(p['likes'] for p in startup_posts) / len(startup_posts)
avg_likes_tech = sum(p['likes'] for p in tech_posts) / len(tech_posts)

if avg_likes_startup > avg_likes_tech * 1.5:
    # Startup content resonates more
    persona_note = "Strongest voice when discussing startup ecosystem"
```
