## Getting Started

### Step 1: Access the Repository

**GitHub:** `https://github.com/jrenaldi79/writing-style`

You can either:
- Clone it: `git clone https://github.com/jrenaldi79/writing-style.git`
- Or just open the README.md file directly on GitHub

### Step 2: Copy & Paste into ChatWise

1. Open ChatWise
2. Start a new chat (use any model with file system access)
3. Open the **README.md** file from the repo
4. Copy **everything below the "PASTE EVERYTHING BELOW" line**
5. Paste it into ChatWise
6. Hit send

### Step 3: Follow the AI's Instructions

The AI will guide you through:
- Cloning the project files to your computer
- Creating your personal workspace
- Setting up your ChatWise assistants
- Analyzing your writing
- Generating your personalized prompt

**You don't need to understand the technical details.** Just follow along and answer the AI's questions.
