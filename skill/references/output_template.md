## Quality Checklist

Before saving, verify:

- [ ] 2+ examples per persona
- [ ] Rules are commands, not descriptions
- [ ] Triggers don't overlap
- [ ] Anti-patterns are specific
- [ ] 800-1500 words total
- [ ] Usable by someone unfamiliar with the user
